Formularios

Estructura basica de un formulario

  • La etiqueta "<form>" engloba el formulario.
  • La etiqueta "<label>" sirve para escribir el nombre del campo a rellenar.
  • La etiqueta "<input>" sirve para crear un campo que permita al usuario interactuar.
  • La etiqueta "<button>" crea un boton que permitira enviar el formulario.

Asociar input y label

El atributo "for" sirve para vincular una etiqueta label a una etiqueta input especifica por medio del id del input.

Muestra de formulario

Codigo:

<form>
   <labelfor="nombre">Label del formulario</label>
   <input id="nombre">
   <button>Boton del formulario</button>
</form>

Tipos de input

Los tipos de input se declaran dentro de la etiqueta input con la propiedad "type".

Inputs de tipo boton

Tipo button

  • Es un boton que se comporta igual que la etiqueta button, pero no sirve para enviar formulario, su funcion sera programada en javascript.
  • La propiedad "type" sirve para declarar el nombre dentro del boton.
Codigo: <input type="button" value="Input button">

Tipo submit

  • Es un boton que se utiliza para enviar el formulario.
Codigo: <input type="submit" value="Input submit">

Inputs de tipo fecha

Tipo date

  • Se utiliza para introducir una fecha.
Codigo: <input type="date">

Tipo time

  • Se utiliza para introducir una hora.
Codigo: <input type="time">

Tipo datetime-local

  • Se utiliza para introducir fecha y hora, no funciona en firefox.
Codigo: <input type="datetime-local">

Tipo week

  • Se utiliza para introducir el numero de semana.
Codigo: <input type="week">

Tipo month

  • Se utiliza para introducir el mes.
Codigo: <input type="month">

Inputs para moviles

Tipo search

  • Se utiliza para las barras de busqueda.
Codigo: <input type="search">

Tipo tel

  • Se utiliza para introducir numeros telefonicos.
Codigo: <input type="tel">

Tipo email

  • Se utiliza para introducir un email.
Codigo: <input type="email">

Tipo password

  • Se utiliza para introducir contrase√±a.
Codigo: <input type="password">

Tipo url

  • Se utiliza para introducir un URL.
Codigo: <input type="url">

Inputs adicionales

Tipo color

  • Se utiliza para especificar un color.
Codigo: <input type="color">

Tipo number

  • Se utiliza para valores numericos.
Codigo: <input type="number">

Tipo range

  • Se utiliza para establecer un rango.
  • El atributo "step" sirve para definir los pasos.
  • El atributo "min" y "max" se utiliza para establecer el rango minimo y maximo respectivamente.
Codigo: <input type="range" step="2" min="0" max="10">

Tipo reset

  • Se utiliza para resetear el formulario.
Codigo: <input type="reset">

Tipo text

  • Valor por defecto.
Codigo: <input type="text">
