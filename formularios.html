Formularios

Inicio

Estructura basica de un formulario

  • La etiqueta "<form>" engloba el formulario.
  • La etiqueta "<label>" sirve para escribir el nombre del campo a rellenar.
  • La etiqueta "<input>" sirve para crear un campo que permita al usuario interactuar.
  • La etiqueta "<button>" crea un boton que permitira enviar el formulario.

Asociar input y label

El atributo "for" sirve para vincular una etiqueta label a una etiqueta input especifica por medio del id del input.

Muestra de formulario

Codigo:

<form>
   <labelfor="nombre">Label del formulario</label>
   <input id="nombre">
   <button>Boton del formulario</button>
</form>

Tipos de input

Los tipos de input se declaran dentro de la etiqueta input con la propiedad "type".

Inputs de tipo boton

Tipo button

  • Es un boton que se comporta igual que la etiqueta button, pero no sirve para enviar formulario, su funcion sera programada en javascript.
  • La propiedad "type" sirve para declarar el nombre dentro del boton.
Codigo: <input type="button" value="Input button">

Tipo submit

  • Es un boton que se utiliza para enviar el formulario.
Codigo: <input type="submit" value="Input submit">

Inputs de tipo fecha

Tipo date

  • Se utiliza para introducir una fecha.
Codigo: <input type="date">

Tipo time

  • Se utiliza para introducir una hora.
Codigo: <input type="time">

Tipo datetime-local

  • Se utiliza para introducir fecha y hora, no funciona en firefox.
Codigo: <input type="datetime-local">

Tipo week

  • Se utiliza para introducir el numero de semana.
Codigo: <input type="week">

Tipo month

  • Se utiliza para introducir el mes.
Codigo: <input type="month">

Inputs para moviles

Tipo search

  • Se utiliza para las barras de busqueda.
Codigo: <input type="search">

Tipo tel

  • Se utiliza para introducir numeros telefonicos.
Codigo: <input type="tel">

Tipo email

  • Se utiliza para introducir un email.
Codigo: <input type="email">

Tipo password

  • Se utiliza para introducir contrase√±a.
Codigo: <input type="password">

Tipo url

  • Se utiliza para introducir un URL.
Codigo: <input type="url">

Inputs adicionales

Tipo color

  • Se utiliza para especificar un color.
Codigo: <input type="color">

Tipo number

  • Se utiliza para valores numericos.
Codigo: <input type="number">

Tipo range

  • Se utiliza para establecer un rango.
  • El atributo "step" sirve para definir los pasos.
  • El atributo "min" y "max" se utiliza para establecer el rango minimo y maximo respectivamente.
Codigo: <input type="range" step="2" min="0" max="10">

Tipo reset

  • Se utiliza para resetear el formulario.
Codigo: <input type="reset">

Tipo text

  • Valor por defecto.
Codigo: <input type="text">

Inputs seleccionables

Tipo radio

  • Permite seleccionar una unica opcion de una lista de opciones relacionadas.
    • Con el atributo "name" podra relacionar a una misma lista de opciones siempre que tengan el mismo valor.
    • Con el atributo booleano "checked" para marcar una opcion por defecto(solo puede tenerlo una).
                                
                                    Codigo: 
                                    <label>Opcion 1<input type="radio"  name="opciones" checked></label>
                                    <label><input type="radio"  name="opciones"></label>

Tipo checkbox

  • Permite seleccionar varias opciones de una lista de opciones relacionadas.
    • Con el atributo "name" podra relacionar a una misma lista de opciones siempre que tengan el mismo valor.
    • Con el atributo booleano "checked" para marcar una opcion por defecto, que pueden ser mas de una.
                                
                                    Codigo: 
                                    <label>Opcion 1<input type="checkbox"  name="opciones" checked></label>
                                    <label><input type="checkbox"  name="opciones"></label checked>

Tipo file

  • Se utiliza para cargar archivos.
                                Codigo: <input type="file">

Tipo select

  • Crea una lista de opciones donde podemos seleccionar una o varias opciones
    • Con el atributo booleano "multiple" se podran seleccionar varias opciones(usando la tecla ctrl+click), este es asignado en la etiqueta "select".
    • Con el atributo "name" podra nombrar la lista de opciones, en la etiqueta "select".
    • Cada opcion ira dentro de una etiqueta "option", donde tambien se asignara el valor del atributo "value".
    • Con la etiqueta "optgroup" se pueden ordenar las opciones por categoria.
                                
                                    Codigo:
                                    <select name="opciones-select" multiple>
                                        <optgroup label="Categoria 1">
                                            <option value="op1">Opcion 1</option>
                                            <option value="op2">Opcion 2</option>
                                        </optgroup>
                                        <optgroup label="Categoria 2">
                                            <option value="op3">Opcion 3</option>
                                        </optgroup>
                                    </select>

Tipo list(con etiqueta datalist)

  • Crea una lista de opciones desplegables que ademas se puede escribir sobre el area vacia para una busqueda con filtros.
    • Necesita un atributo "list" para hacer referencia a la eitqueta "datalist" que contendra todas las opciones.
    • La etiqueta "datalist" necesita el atributo "id" con el mismo valor del atributo "list" para ser asignado por el input.
    • Con la etiqueta "option" declara cada una de las opciones.
    • Con el atributo "value" se inserta el nombre de la opcion, por lo que no es necesario declararlo dentro de la etiqueta "option"
                                
                                Codigo:
                                <input type="list" list="datalist-1">
                                <datalist id="datalist-1">
                                    <option value="Opcion 1">No es necesario declarar un valor en esta parte.</option>
                                    <option value="Opcion 2">Puede servir para asignar una descripcion.</option>
                                    <option value="Opcion 3"></option>
                                </datalist>

Etiquetas de formulario

  • Ademas de las etiquetas para la estructura basica de un formulario, tambien existen otras adicionales.

Etiqueta fieldset

  • Se utiliza para agrupar elementos dentro de un formulario.

    • Etiqueta legend

    • Representa una etiqueta para el contenido del fieldset.
Legend de fieldset 1
Legend de fieldset 2 
                            
                                <fieldset>
                                    <legend>Legend de fieldset 1</legend>
                                    <label for="uno">Primer elemeno</label>
                                    <input type="text" id="uno">
                                    <label for="dos">Segundo elemento</label>
                                    <input type="text" id="dos">
                                </fieldset>
                                <fieldset>
                                    <legend>Legend de fieldset 2</legend>
                                    <label for="tres">Tercer elemeno</label>
                                    <input type="text" id="tres">
                                    <label for="cuatro">Cuarto elemento</label>
                                    <input type="text" id="cuatro">
                                </fieldset>

Etiqueta meter

  • Representa un valor dentro de un rango conocido.
    • Dependdiendo del rango del atributo "value" se puede repersentar un valor segun los atributos "low","optimum" y "high".
                            Codigo:
                            <label for="fuel-1">Rango</label>
                            <meter id="fuel-1" value="20" min="0" max="100" low="30" optimum="50" high="80"></meter>
                            <label for="fuel-2">Rango 2</label>
                            <meter id="fuel-2" value="50" min="0" max="100" low="30" optimum="50" high="80"></meter>
                            <label for="fuel-3">Rango 3</label>
                            <meter id="fuel-3" value="81" min="0" max="100" low="30" optimum="50" high="80"></meter>

Etiqueta progress

  • Representa el progreso de una tarea.
                            
                                <label for="task">Tarea</label>
                                <progress id="task" value="80" max="100"></progress>

Etiqueta textarea

  • Se utiliza para introducir texto en un formulario.
                            
                                <label for="text">Mensaje</label>
                                <textarea id= "text" cols="100" rows="2">Escribe el mensaje aqui.</textarea>

Atributos de los formularios

Atributo placeholder

  • Da una pista de lo que el usuario tiene que escribir.
                            
                                Codigo:
                                <label for="nombre-1">Nombre: </label>
                                <input type="text" id="nombre-1" placeholder="Introduce tu nombre">

Atributo required

  • Hace que un campo sea obligatorio.
  • En caso de que no se rellene el campo, al momento de enviar el formulario saldra un mensaje que advierte al usuario.
                            
                                Codigo:
                                <label for="camp-oblig">Apellido: </label>
                                <input type="text" id="camp-oblig" required>
                                <button>Enviar formulario</button>

Atributo readonly

  • Hace que un campo sea de solo lectura.
                            
                                Codigo:
                                <label for="camp-readonly">Campo de lectura: </label>
                                <input type="text" id="camp-readonly" value="No se puede modificar." readonly>

Atributo disable

  • Desactiva el campo para que no se puede modificar.
  • Este campo no se enviara junto con el formulario.
                            
                                Codigo:
                                <label for="atr-dis">Campo desactivado: </label>
                                <input type="text" id="atr-dis" disabled>

Atributo min y max

  • Establece el minimo y maximo de un campo numerico.
  • En caso de que el valor no este dentro del rango, saldra un mensaje de advertencia para el usuario, por lo que no podra enviar el formulario.
                            
                                Codigo:
                                <label for="min-max">Campo numerico: </label>
                                <input type="number" min="2" max="10">
                                <button>Enviar formularo</button>

Atributo minlength y maxlength

  • Establece el minimo y maximo de caracteres de un campo de texto.
  • En caso de que el texto no este dentro del rango, saldra un mensaje de advertencia para el usuario, por lo que no podra enviar el formulario.
                            
                                Codigo:
                                <label for="minl-maxl">Introduce texto: </label>
                                <input type="text" id="minl-maxl" minlength="2" maxlength="10">
                                <button>Enviar formulario</button>

Atributo selected

  • Sirve para establecer una opcion por defecto, similar al atributo checked.
                            
                                Codigo:
                                <label for="selec">Selecciona una opcion: </label>
                                <select name="opciones" id="selec">
                                    <option value="opcion-01">Opcion 1</option>
                                    <option value="opcion-02" selected>Opcion 2</option>
                                </select>

Atributo autofocus

  • Enfoca el campo por defecto al cargar el formulario.
                            
                                Codigo:
                                <label for="focus-tx">Campo enfocado: </label>
                                <input type="text" id="focus-tx" autofocus>
