Etiquetas de bloque

  • La etiqueta "adress" se utiliza para aportar informacion de contacto.
    • colonia
    calle
    codigo postal
  • La etiqueta "blockquote" se utiliza para marcar la cita a otros autores.Puede incluir el atributo "cite" para indicar el enlace a la fuente original.
    • Ser o no ser esa es la cuestion.
  • La etiqueta "pre" se utiliza para tener codigo preformateado que necesita ser representado igual que se escribio.
    • El contenido se representa igual como esta escrito en...           html.
  • La etiqueta "div" se utiliza para dividir el contenido, sobretodo para dar estilos con css.

    • Contenido dentro de una etiqueta div.

  • La etiqueta "hr" se utiliza para decirle al navegador un cambio de tema.
