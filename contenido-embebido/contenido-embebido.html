Que es el contenido embebido?

  • Es todo contenido que nos traemos desde fuera.Son los archivos que mas peso añaden a un sitio web.
  • Los tipos mas conocidos son:
    • Imagenes
    • Audio
    • Videos
    • Iframes

Imagenes

  • Los formatos de imagenes para web se clasifican en dos tipos:
    • Vectoriales:
      • "svg"(recomendado) no pierde calidad al hacer zoom y no tiene peso en la web.
    • Mapa de bits:
      • "jpg"
      • "png" 8 y 24 para transparencia
      • "gif" para animaciones de imagen.
      • "webp"(recomendado) pesa muy poco, soporta conversiones de jpg, png y gif.

Insertar imagenes en HTML

Etiqueta img

  • Se utiliza para cargar una imagen, el atributo "src" tiene la ruta del archivo.
Codigo:
 <img src="/contenido-embebido/assets/imagenes/logo-de-facebook.svg" alt="Logo de facebook" class="img-logo-1">

Device pixel ratio

DPR(Pixeles reales / Pixeles disponibles)
Es la relacion entre los pixeles reales que tiene el dispositivo y los pixeles que tenemos disponibles para pintar contenido(viewport).
Pagina para ver el DPR.

Atributo srcset

  • Sirve para tener varias imagenes(con diferentes medidas,resoluciones), y asi cargar el archivo que se adapte mejor al dispositivo.
  • Dependiendo del DPR sera la imagen que se pinte para el usuario.
    • La forma de declarar el DPR sera con Nx, siendo"N" el valor del DPR y "x" una constante.
Codigo:
<img srcset="/contenido-embebido/assets/imagenes/facebook-128px.png,
/contenido-embebido/assets/imagenes/facebook-64px.png x2"
alt="logo-de-facebook-2" class="img-logo-2">

Etiqueta picture

  • Es un contenedor usado para especificar múltiples elementos "source" y un elemento "img" contenido en él para proveer versiones de una imagen para diferentes escenarios de dispositivos.
    • La etiqueta "source" especifica recursos de medios múltiples para los elementos(imagenes, audio, video).
      • El atributo "media" declara una condicion para cargar la imagen.
                        
                            Codigo:
                            <picture>
                                <source srcset="/contenido-embebido/assets/imagenes/facebook-64px.webp" media="(max-width:1200px)">
                                <img src="/contenido-embebido/assets/imagenes/facebook-128px.png" alt="logo-por-defecto">
                            </picture>

Insertar audio en HTML

  • La etiqueta "audio" declara el audio a reproducir.
    • El atributo booleano "controls" muestra los controles de audio al usuario, es obligatorio.
    • El atributo booleano "autoplay" reproduce el audio automaticamente al entrar en la pagina.
    • El atributo booleano "muted" mutea el auto por defecto.
    • El atributo booleano "loop" se crea un bucle que reproduce el audio de forma indefinida.

Insertar video en HTML

  • La etiqueta video declara el video a reproducir.
    • Tiene los mismos atributos que la etiqueta "audio", mas un atributo adicional.
    • Con el atributo "poster" se pinta una imagen por defecto antes de reproducir el video.
                        
                            Codigo:
                            <video src="./assets/videos/Hina2.mp4" controls muted loop poster="./assets/imagenes/cursor-1920x1080.png" class="videos">
