Imagenes

  • Los formatos de imagenes para web se clasifican en dos tipos:
    • Vectoriales:
      • "svg"(recomendado) no pierde calidad al hacer zoom y no tiene peso en la web.
    • Mapa de bits:
      • "jpg"
      • "png" 8 y 24 para transparencia
      • "gif" para animaciones de imagen.
      • "webp"(recomendado) pesa muy poco, soporta conversiones de jpg, png y gif.

Insertar imagenes en el HTML

Etiqueta img

  • Se utiliza para cargar una imagen, el atributo "src" tiene la ruta del archivo.
Codigo:
 <img src="/contenido-embebido/assets/imagenes/logo-de-facebook.svg" alt="Logo de facebook" class="img-logo-1">

Device pixel ratio

DPR(Pixeles reales / Pixeles disponibles)
Es la relacion entre los pixeles reales que tiene el dispositivo y los pixeles que tenemos disponibles para pintar contenido(viewport).
Pagina para ver el DPR.

Atributo srcset

  • Sirve para tener varias imagenes(con diferentes medidas,resoluciones), y asi cargar el archivo que se adapte mejor al dispositivo.
  • Dependiendo del DPR sera la imagen que se pinte para el usuario.
    • La forma de declarar el DPR sera con xN, siendo "x" constante y "N" el valor del DPR.
Codigo:
<img srcset="/contenido-embebido/assets/imagenes/facebook-128px.png,
/contenido-embebido/assets/imagenes/facebook-64px.png x2"
alt="logo-de-facebook-2" class="img-logo-2">
