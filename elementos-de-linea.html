Elementos de linea

Inicio

Etiquetas: em, strong, small, br, wbr, time, i, b, u, sup, sub

  • La etiqueta "em" sirve para dar enfasis a una palabra.

  • La etiqueta "strong" sirve para dar mas enfasis.

  • La etiqueta "small" sirve para dar menos enfasis que el texto normal.

  • La etiqueta "br" sirve para forzar
    un salto de linea.

    • a etiqueta "wbr" sirve para hacer un salto de linea solo si hiciera falta.

  • La etiqueta "time" sirve para representar el contenido de hora/fecha. 16/08/2021 11:17

  • La etiqueta "i" sirve para dar estilo italico.

  • La etiqueta "b" sirve para dar estilo bold.

  • La etiqueta "u" sirve para dar estilo underline.

  • La etiqueta "sup" sirve para dar estilo superindice.

  • La etiqueta "sub" sirve para dar estilo subindice.

Etiquetas: span, q, code, codigo ASCII

  • La etiqueta "span" se usa como contenedor, parecido a la etiqueta div.

  • La etiqueta "q" se usa para citar una frase, parecido a la etiqueta blockquote.

  • La etiqueta "code" se usa para representar codigo y que se visualize tal cual esta escrito.

                                
                                span{
                                    color: red;
                                }

  • El codigo ASCII se usa para representar caracteres y simbolos especiales a traves de html.

                                <h1> </h1>
