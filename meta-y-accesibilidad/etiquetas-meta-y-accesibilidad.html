Etiquetas meta y accesibilidad

Inicio

Etiqueta meta

  • Proporciona instrucciones para la indexación en los buscadores web.
  • Se deben escribir dentro de la etiqueta "head".
                    
                        Codigo:
                        <head>
                        <meta charset="UTF-8">
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <meta name="description" content="Texto que sera mostrado en el buscador de google.">
                        <title>Curso de html 2020:accesiblidad</title>
                        </head>

Favicons

  • Es la imagen que se muestra en la pestaña del navegador.
  • Se usa una etiqueta "link" para declarar el elemento.
  • Debe estar dentro de la etiqueta "head".
                    
                        Codigo:
                        <link rel="apple-touch-icon" sizes="57x57" href="./assets/meta/favicons/apple-icon-57x57.png">
Pagina recomendada para convertir una imagen en un set de favicons para diferentes navegadores: https://www.favicon-generator.org/

Uso de Fontawesome

  • Libreria de iconos, una de las mas usada en el mundo.
  • Primero se pega la etiqueta link de la libreria dentro de la etiqueta "head" para poder usarla.
  • Y despues se pega el codigo del icono que quieras dentro de la pagina.
Campana 
                    
                        Codigo:
                        <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.15.4/css/all.css" integrity="C6Up4h/KRBcigm" crossorigin="anonymous">
                        <i class="far fa-bell"> Campana</i>

Accesibilidad

  • Es la práctica inclusiva de garantizar que el mayor numero de personas puedan usarlo, incluyendo las personas con discapacidad.

Atributo tabindex

  • Permite al usuario moverse a traves de la pagina siguiendo un orden,usando la tecla tab.

                                    Codigo:
                                    <p tabindex="1">Elemento 1</p>
                                    <p tabindex="2">Elemento 2</p>
                                    <p tabindex="3">Elemento 3</p>

Atributo aria-label

  • Sirve para indicar al navegador la informacion que debe leer(usando screen reader).
Leer mas 
                                
                                    Codigo:
                                    <a href="#" aria-label="Leer mas sobre el tema de programacion web 2020">Leer mas</a>

Atributo role

  • Sirve para indicarle al navegador sobre que tipo de elemento nos estamos posicionando.
Leer mas 
                                
                                    Codigo:
                                    <a href="#" role="link">Leer mas</a>

Protocolo Open Graph

  • Es un protocolo creado por Facebook para poder elegir que se mostrara en una publicacion cuando se comparta un link.
                            
                                Codigo:
                                <meta property="og:title" content="The Rock" />
                                <meta property="og:type" content="video.movie" />
                                <meta property="og:url" content="https://www.imdb.com/title/tt0117500/" />
                                <meta property="og:image" content="https://ia.media-imdb.com/images/rock.jpg" />
